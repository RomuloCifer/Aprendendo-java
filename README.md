# Java

Este repositório reúne exercícios e pequenos projetos em Java que venho fazendo para estudar e praticar programação. O foco está em consolidar os fundamentos: lógica, estruturas básicas, manipulação de strings, loops, arrays e outros conceitos essenciais da linguagem.

## Estrutura

Cada pasta contém um mini-projeto ou exercício independente:

- **caixinhaDeSon/** - Projeto relacionado a áudio/som
- **ChecarProteinaEmDNA/** - Verificação de proteínas em sequências de DNA
- **FizzBuzz/** - Implementação do clássico FizzBuzz
- **metodopagamento/** - Sistema de pagamentos com POO (classes abstratas, polimorfismo e herança)
- **numeroPrimo/** - Verificação de números primos
- **Polimorfismo/** - Exemplos de polimorfismo com animais
- **SimuladorDado/** - Simulador de lançamento de dados com estatísticas

## Sobre

Estes projetos fazem parte do meu processo de aprendizado contínuo em Java. O código está em constante evolução conforme avanço nos estudos.
